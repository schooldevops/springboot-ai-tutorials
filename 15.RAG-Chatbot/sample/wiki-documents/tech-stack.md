# 기술 스택

## 백엔드

### 언어 및 프레임워크

- **Kotlin**: 주 개발 언어
- **Spring Boot 3.x**: 웹 애플리케이션 프레임워크
- **Spring Data JPA**: ORM 및 데이터 액세스
- **Spring Security**: 인증 및 권한 관리
- **Spring AI**: AI 기능 통합

### 데이터베이스

- **PostgreSQL 15**: 메인 데이터베이스
- **Redis**: 캐싱 및 세션 관리
- **MongoDB**: 로그 및 비정형 데이터 저장

### 메시징

- **Kafka**: 이벤트 스트리밍
- **RabbitMQ**: 메시지 큐

## 프론트엔드

### 웹

- **React 18**: UI 라이브러리
- **TypeScript**: 타입 안전성
- **Next.js**: 서버 사이드 렌더링
- **TailwindCSS**: 스타일링
- **React Query**: 데이터 페칭

### 모바일

- **Flutter**: 크로스 플랫폼 모바일 앱
- **Kotlin Multiplatform**: 네이티브 모듈

## DevOps

### 컨테이너 및 오케스트레이션

- **Docker**: 컨테이너화
- **Kubernetes**: 컨테이너 오케스트레이션
- **Helm**: Kubernetes 패키지 관리

### CI/CD

- **GitHub Actions**: CI/CD 파이프라인
- **ArgoCD**: GitOps 기반 배포
- **SonarQube**: 코드 품질 분석

### 모니터링

- **Prometheus**: 메트릭 수집
- **Grafana**: 시각화
- **ELK Stack**: 로그 관리
  - Elasticsearch
  - Logstash
  - Kibana
