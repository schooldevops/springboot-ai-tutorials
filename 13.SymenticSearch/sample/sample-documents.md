# 샘플 문서 데이터

## 기술 문서

### 프로그래밍 언어

```
Kotlin은 JetBrains에서 개발한 현대적인 프로그래밍 언어입니다. JVM, Android, JavaScript, Native 플랫폼을 지원하며, Java와 100% 상호운용이 가능합니다. Null 안전성, 확장 함수, 코루틴 등의 기능을 제공합니다.
```

```
Java는 객체지향 프로그래밍 언어로, "Write Once, Run Anywhere" 철학을 가지고 있습니다. JVM에서 실행되며, 엔터프라이즈 애플리케이션 개발에 널리 사용됩니다.
```

```
Python은 간결하고 읽기 쉬운 문법을 가진 고급 프로그래밍 언어입니다. 데이터 분석, 머신러닝, 웹 개발 등 다양한 분야에서 활용됩니다.
```

```
TypeScript는 JavaScript의 상위 집합으로, 정적 타입 시스템을 제공합니다. 대규모 웹 애플리케이션 개발에 적합하며, Angular, React 등과 함께 사용됩니다.
```

### 프레임워크

```
Spring Boot는 Java 기반의 웹 애플리케이션 프레임워크입니다. 자동 설정, 내장 서버, 프로덕션 준비 기능을 제공하여 빠른 개발을 가능하게 합니다. RESTful API, 마이크로서비스 개발에 최적화되어 있습니다.
```

```
React는 Facebook에서 개발한 사용자 인터페이스 구축을 위한 JavaScript 라이브러리입니다. 컴포넌트 기반 아키텍처와 가상 DOM을 사용하여 효율적인 UI 렌더링을 제공합니다.
```

```
Django는 Python 기반의 웹 프레임워크로, "배터리 포함" 철학을 따릅니다. ORM, 관리자 페이지, 인증 시스템 등을 기본 제공하여 빠른 웹 개발을 지원합니다.
```

```
Express.js는 Node.js를 위한 미니멀리스트 웹 프레임워크입니다. 가볍고 유연하며, RESTful API 서버 구축에 널리 사용됩니다.
```

### 데이터베이스

```
PostgreSQL은 오픈소스 관계형 데이터베이스 관리 시스템입니다. ACID 트랜잭션, 복잡한 쿼리, JSON 지원 등 강력한 기능을 제공합니다. 확장성과 표준 준수로 유명합니다.
```

```
MongoDB는 NoSQL 문서 지향 데이터베이스입니다. JSON과 유사한 BSON 형식으로 데이터를 저장하며, 스키마가 유연하고 수평적 확장이 용이합니다.
```

```
Redis는 인메모리 키-값 저장소입니다. 캐싱, 세션 관리, 실시간 분석 등에 사용되며, 매우 빠른 읽기/쓰기 성능을 제공합니다.
```

```
MySQL은 세계에서 가장 널리 사용되는 오픈소스 관계형 데이터베이스입니다. 웹 애플리케이션의 백엔드 데이터 저장소로 많이 활용됩니다.
```

### DevOps 도구

```
Docker는 컨테이너 기반 가상화 플랫폼입니다. 애플리케이션과 의존성을 컨테이너로 패키징하여 어디서나 동일하게 실행할 수 있습니다. 마이크로서비스 아키텍처에 필수적입니다.
```

```
Kubernetes는 컨테이너 오케스트레이션 플랫폼입니다. 컨테이너화된 애플리케이션의 배포, 스케일링, 관리를 자동화합니다. 클라우드 네이티브 애플리케이션의 표준입니다.
```

```
Jenkins는 오픈소스 CI/CD 도구입니다. 빌드, 테스트, 배포 파이프라인을 자동화하여 개발 생산성을 높입니다.
```

```
GitHub Actions는 GitHub에 통합된 CI/CD 플랫폼입니다. 워크플로우를 YAML로 정의하여 자동화된 빌드, 테스트, 배포를 수행합니다.
```

### 클라우드 서비스

```
AWS (Amazon Web Services)는 세계 최대의 클라우드 컴퓨팅 플랫폼입니다. EC2, S3, RDS, Lambda 등 200개 이상의 서비스를 제공합니다.
```

```
Google Cloud Platform (GCP)은 Google의 클라우드 인프라를 활용한 서비스입니다. BigQuery, Kubernetes Engine, Cloud Functions 등을 제공합니다.
```

```
Microsoft Azure는 Microsoft의 클라우드 플랫폼입니다. Windows 기반 애플리케이션과의 통합이 우수하며, AI/ML 서비스가 강력합니다.
```

```

```

```
휴가 정책: 연차는 입사 1년 후부터 15일이 부여되며, 3년 이상 근속 시 매년 1일씩 추가됩니다. 최대 25일까지 누적 가능합니다.

```


### FAQ

```
Q: 출장 시 경비 처리는 어떻게 하나요?
A: 출장 전 출장 신청서를 제출하고, 귀사 후 3일 이내에 영수증과 함께 경비 정산서를 제출하면 됩니다.
```

```
Q: 사내 교육 프로그램은 어떤 것이 있나요?
A: 기술 세미나, 외부 컨퍼런스 참가 지원, 온라인 강의 구독, 사내 스터디 그룹 등이 있습니다.
```

```
Q: 장비 지원은 어떻게 받나요?
A: 입사 시 노트북, 모니터, 키보드, 마우스가 기본 지급됩니다. 추가 장비가 필요한 경우 팀장 승인 후 구매 가능합니다.
```
